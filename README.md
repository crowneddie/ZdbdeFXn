# 项目简介

## 核心功能
- 提供基于YOLOv8的牛羊目标检测系统，支持对牛(cattle)和羊(sheep)两类目标的识别
- 支持多种输入源检测：单张图片、图片文件夹、视频文件及本地摄像头
- 提供用户界面，可实时显示目标位置、目标总数和置信度信息
- 支持自定义置信度和IOU阈值参数
- 提供批量检测功能，可在界面直接查看所有检测结果
- 支持检测结果保存功能
- 包含完整源码、3300张标注数据集、训练好的模型及训练结果、项目运行教程

## 适用场景
- 农牧业中牛羊数量统计与监测
- 畜牧养殖场自动化管理
- 农业科研中的动物行为研究
- 畜牧业智能化管理系统的开发参考

## 技术特点
- 基于Python 3.9开发，依赖OpenCV-Python、PyQt5、Ultralytics和Torch1.9等框架
- 采用YOLOv8目标检测算法，具有较高的检测精度和速度
- 提供图形用户界面，操作简便直观
- 系统已配置完整环境，可直接运行使用
- 包含完整的训练数据集，支持进一步模型优化和二次开发
